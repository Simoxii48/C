1-Convert Octal To Decimal

	-100
		0 * 8^0 = 0 * 1 = 0
		0 * 8^1 = 0 * 8 = 0
		1 * 8^2 = 1 * 64 = 64
					100(8) >>> 64(10)

	-512
		2 * 8^0 = 2 * 1 = 2
		1 * 8^1 = 1 * 8 = 8
		5 * 8^2 = 5 * 64 = 320
					512(8) >>> 330(10)

2- Convert Decimal To Octal

	-64	
		64 / 8 = 8 >>> 0
		8  / 8 = 1 >>> 0
		1  / 8 = ----- 1
					64(10) >>> 100(8)

	-330
		330 / 8 = 41.25 >>> 0.25 * 8 = 2
		41  / 8 = 5.125 >>> 0.125 * 8 = 1
		5   / 8 = ------ 5
					330(10) >>> 512(8)

3-Convert Direct Octal To Binary

	-100
		1	0	0
		001	000	000	
					100(8) >>> 0100 0000(2)

	-512
		5	1	2
		101	001	010
					512(8) >>> 0001 0100 1010(2)

4-Convert Direct Binary To Octal

	-0100 0000
		001	000	000
		1	0	0
 					0100 0000(2) >>> 100(8)

	-0001 0100 1010
		000	101	001	010
		0	5	1	2
					0001 0100 1010(2) >>> 512(8)
